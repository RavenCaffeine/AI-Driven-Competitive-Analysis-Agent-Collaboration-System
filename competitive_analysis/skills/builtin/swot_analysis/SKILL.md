# SWOT Analysis
description: Deep SWOT analysis skill for competitive intelligence

Perform a comprehensive SWOT (Strengths, Weaknesses, Opportunities, Threats) analysis.

## Instructions
Given competitor data, generate a detailed SWOT analysis with:
- Each item backed by evidence
- Impact rating (high/medium/low)
- Source references for traceability

## Output Format
Return structured JSON conforming to the SWOTAnalysis schema.
